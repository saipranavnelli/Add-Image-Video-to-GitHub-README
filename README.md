# 📸 How To Add Image To GitHub README

## 📖 Introduction

Today, I'll guide you through how to add an image to a GitHub README file.

If you've been searching for a way to add a screenshot or an image into your GitHub `README.md` file, you've come to the right place.

Adding an image to a GitHub README file helps improve documentation by including visuals like screenshots, logos, and diagrams.

---

## 🧠 What You Will Learn

- How to upload images to GitHub
- How to get the image URL
- How to display images in README.md
- Best practices for using images

---

## 🔶 Steps To Add Image To GitHub README

### 1️⃣ Open GitHub
Go to 👉 https://github.com

---

### 2️⃣ Open Your Repository
Select the repository where you want to add the image.

---

### 3️⃣ Upload Image
- Click on **Add file**
- Select **Upload files**
- Drag and drop your image

---

### 4️⃣ Commit Changes
Click on **Commit changes**

---

### 5️⃣ Open Image
Click on the uploaded image from your repository

---

### 6️⃣ Copy Image Link
Right-click → Copy image address

---

### 7️⃣ Edit README
Click on ✏️ (Edit README)

---

![Alt Text](image_url)
